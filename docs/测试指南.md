#### 🖥️ 如何启动和测试

##### 2. 启动前端应用

#### 🔧 常见问题解决

##### 端口冲突
如果遇到端口冲突：
```bash
# 查看占用端口的进程
netstat -ano | findstr :8000
netstat -ano | findstr :3000

# 终止进程
taskkill /F /PID [进程ID]
```

##### 依赖安装问题
```bash
# 后端依赖
cd F:/claude_projects/cvStudio/backend
pip install -r requirements.txt

# 前端依赖
cd F:/claude_projects/cvStudio/frontend
npm install
```

##### 数据库问题
如果遇到数据库问题：
```bash
# 重新创建测试用户
cd F:/claude_projects/cvStudio/backend
python create_test_user.py
```
