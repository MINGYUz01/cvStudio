### 🐛 故障排除

#### 如果登录失败：
1. 检查后端是否在运行：http://localhost:8000
2. 检查前端是否在运行：http://localhost:3000
3. 检查用户名和密码是否正确
4. 查看浏览器控制台是否有错误信息

#### 如果端口冲突：
```bash
# 查看端口占用
netstat -ano | findstr :8000
netstat -ano | findstr :3000

# 终止进程
taskkill /F /PID [进程ID]
```
